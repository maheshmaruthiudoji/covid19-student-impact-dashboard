🦠 Impact of COVID-19 Dashboard
An Interactive Power BI Dashboard for Analyzing the Impact of COVID-19 on Students' Lifestyle, Mental Health, and Daily Activities

The Impact of COVID-19 Dashboard is an interactive Power BI report that analyzes how the COVID-19 pandemic affected students' daily routines, including online learning, sleep patterns, social media usage, fitness, self-study habits, eating behavior, and stress management. The dashboard provides meaningful insights that can help educators, researchers, and policymakers understand the lifestyle changes experienced by students during the pandemic.

📌 Short Description / Purpose

The Impact of COVID-19 Dashboard is a data visualization solution built using Power BI to explore the effects of the COVID-19 pandemic on students' academic and personal lives. Through interactive charts and KPIs, the dashboard highlights behavioral changes, study patterns, health habits, and stress-relief activities, enabling stakeholders to make informed educational and wellness decisions.

🛠 Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Primary platform used to build interactive reports and dashboards.
📂 Power Query Editor – Used for cleaning, transforming, and preparing the dataset.
🧠 DAX (Data Analysis Expressions) – Developed calculated measures, KPIs, and dynamic calculations.
🔗 Data Modeling – Established relationships between student demographic and activity datasets.
📈 Interactive Visualizations – KPI Cards, Line Charts, Bar Charts, Donut Charts, Column Charts, and Slicers.
📁 File Formats – .pbix for development and .png for dashboard presentation.
📂 Data Source

Source: Student Lifestyle and COVID-19 Impact Dataset

The dataset contains survey-based information about students during the COVID-19 pandemic, including:

Student Age
Region
Weight Change
Online Class Duration
Sleep Time
Fitness Activity
Social Media Usage
Self-Study Time
Meals Per Day
Stress Busters
Social Media Platforms

The data was cleaned, transformed, and modeled using Power Query before creating interactive visualizations in Power BI.

🌟 Features / Highlights
• Business Problem

The COVID-19 pandemic significantly changed students' lifestyles, education, and mental well-being. Educational institutions and researchers often struggle to analyze large survey datasets and answer questions such as:

How much time do students spend in online classes?
Has student weight changed during the pandemic?
Which stress-relief activities are most common?
How has social media usage changed?
Are students getting enough sleep?
How much time is dedicated to self-study?
What are the differences between regions?

Without an interactive dashboard, identifying these trends becomes difficult and time-consuming.

• Goal of the Dashboard

The dashboard was developed to provide an interactive platform that:

Monitors students' lifestyle changes during COVID-19.
Analyzes online learning and study habits.
Examines sleep and fitness patterns.
Identifies common stress-relief activities.
Compares behavioral differences across regions.
Supports research and educational decision-making.
📊 Walkthrough of Key Visuals
🔹 KPI Cards

Provides a quick overview of important student activity metrics:

Average Time Spent on Online Classes
Average Time Spent on Fitness Activities
Average Sleep Duration
Average Time Spent on Social Media

These KPIs help users quickly understand overall behavioral trends.

🔹 Weight Change Filter

An interactive slicer allowing users to analyze data based on:

Decreased Weight
Increased Weight
Weight Remained Constant

This enables comparison of lifestyle behaviors across different weight-change groups.

🔹 Number of Meals Per Day by Student (Area Chart)

Shows the relationship between students and the number of meals consumed per day.

This visualization helps identify eating habit trends during the pandemic.

🔹 Number of Students by Stress Busters (Horizontal Bar Chart)

Displays the most common stress-relief activities among students, including:

Listening to Music
Online Gaming
Watching Movies
Reading Books
Scrolling Social Media
Sleeping
Online Surfing

This helps understand how students managed stress during lockdown periods.

🔹 Sleep Time by Age (Column Chart)

Analyzes total sleep duration across different student age groups.

This visual helps identify age groups with higher or lower sleep durations and highlights changes in sleeping habits.

🔹 Average Time Spent on Social Media (Bar Chart)

Compares average usage across popular platforms such as:

Instagram
YouTube
WhatsApp

This provides insights into the most frequently used social media platforms during the pandemic.

🔹 Self-Study Time by Region (Donut Chart)

Compares study time between different residential regions:

Delhi-NCR
Outside Delhi-NCR

This helps analyze regional differences in self-study habits.

🔹 Time Spent on Online Classes (Line Chart)

Displays online class duration across different student ages.

This visualization highlights trends in online learning participation and helps identify age groups spending more time in virtual education.

📈 Business Impact & Insights

Student Wellness Monitoring

Provides insights into sleep, fitness, eating habits, and stress management, supporting initiatives that improve student well-being.

Mental Health Awareness

Digital Behavior Analysis

Shows how students use social media during the pandemic, helping institutions promote balanced digital engagement.

Regional Comparison

Highlights differences in self-study habits between regions, supporting localized educational policies and resource allocation.

Academic Performance Support

Analyzing online class participation and self-study patterns helps institutions identify opportunities to improve student engagement and learning outcomes.
